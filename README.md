# idm-rpc
这一坨玩意儿是ai生成的(但居然能用),目前该项目只支持简体中文

如果你憎恨ai生成的东西请离开,我也无意打扰你

能使idm适配上rpc下载,就像aria2那样的rpc下载,且不会干扰idm原本的功能

## 正确配置并使用
首先确保您的电脑上安装了Python

安装依赖库flask

如你所见这个有一个rpc文件夹,将它下载下来并放到 idm 的主程序文件夹中

有一个名为start.bat文件用于开启该项目,您可以创建快捷方式放置于桌面中以便启动

另外有一个名为config.json的配置文件,您需要打开并配置它

第1项用于配置idm主程序的位置(需要注意的是需使用两个斜杠,可参考内置的默认配置文件)

第2项用于配置端口

第3项用于配置主机域名

第4项是调试模式，将它设为true程序将会输出详细的报错信息和一些额外的信息,如果不想看到可以设为false

第5项是密钥,若不想使用留空

第6项是密钥使用状态

第7项是下载位置

第8项是同时最大下载数

第9项是超时时间(以秒数为单位)

第10项是任务历史记录保留天数

第11项和第5项一样是密钥不想使用留空

第12项是日志层级
